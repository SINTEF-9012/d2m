Welcome to d2m - Data to Model

Transforming Raw Data into Dynamic Models

In the evolving landscape of machine learning, "d2m" stands as a pivotal tool designed for practitioners and researchers. Developed at a leading research institution, "d2m" embodies the commitment to open-source innovation, offering a robust and flexible machine learning pipeline tailored for tabular and time series data.

Key Features of d2m:

  • Seamless Transition from Data to Model: d2m streamlines the process of transforming raw data into fully trained machine learning models.
  • Explainable AI: Incorporating tools like SHAP and LIME, d2m ensures that AI decisions are transparent and understandable.
  • Uncertainty Estimation: Utilizing Bayesian dropout techniques, it provides reliable uncertainty estimates for neural network predictions.
  • Green AI: Through integration with CodeCarbon, d2m actively tracks and reports on carbon emissions, promoting environmentally responsible AI development.

Designed for Flexibility and Efficiency:

d2m leverages Data Version Control (DVC) for efficient pipeline orchestration, minimizing redundant runs and maximizing productivity. Whether you're a seasoned data scientist or a budding AI enthusiast, d2m provides the tools you need to create, evaluate, and deploy AI models responsibly and sustainably.

Join the Open Source Movement:

As an open-source platform, d2m welcomes contributions and collaboration from the global community. Together, we're not just building software; we're fostering a movement towards more ethical, sustainable, and transparent AI.

Core Features

d2m is not just a tool; it's a comprehensive solution for modern AI challenges. Here's what sets it apart:

  • Explainable AI: Understand the 'why' behind model decisions with integrated SHAP and LIME explanations.

  • Carbon Emission Tracking: Commit to sustainable AI with CodeCarbon for monitoring and reporting AI's carbon footprint.

  • Data Version Control: Streamline workflow and avoid redundancy with DVC for efficient pipeline management.
